# Title: Stock Predication via Sentiment Analysis

# Project Description
For this project, we will train Neural Networks (NN) to predict the stock prices of several 
different companies. The model will be given the name of a fortune 500 company, and using sentiment 
analysis based on keywords found on social media, the model will give a confidence score 
that will suggest whether to buy or pass. If no dataset exists that contains postive and negative 
labeled words then a dataset for the purpose of sentiment analysis will have to be created.
	
If the project is successful, then future extensions to this project would include looking into more 
volatile stocks, or even expanding the usage of the model to include cryptocurrency as well. Furthermore, 
this project will also look into the ethical considerations of using models to influence the purchase of a company's stock. 
	
# Project Goals
1. Create a model that can uses sentiment analysis to influence its decision.
2. Expand the model to riskier stocks/investments.
3. Train a model that can generate high confidence scores.
